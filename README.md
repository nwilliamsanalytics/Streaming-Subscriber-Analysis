# Streaming Subscriber Analysis

Analysis of subscriber strength, retention, and lifetime value across streaming platforms.

![Subscriber Strength Dashboard](Subscriber%20Strength%20v2.jpg)

---

## Business Question
What actually drives subscriber value in streaming — scale or retention?

---

## Key Insight
Subscriber scale alone does not drive value.

Platforms with similar scale (e.g., Disney+ and HBO Max) generate comparable LTV, while lower churn consistently correlates with higher subscriber value.

---

## Implication
Long-term value in subscription businesses is driven more by retention than by subscriber growth.

---

## What I Built
- Simulated subscriber dataset using Python
- Modeled Lifetime Value (LTV) based on ARPU and churn assumptions
- Created comparative analysis across major streaming platforms
- Visualized insights in Tableau dashboard
